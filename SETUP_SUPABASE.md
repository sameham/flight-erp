# 🔧 إعداد Supabase (مرة واحدة فقط)

## 1) إنشاء المشروع
1. ادخل على **supabase.com** واعمل حساب مجاني (تقدر بحساب GitHub)
2. اضغط **New Project** → اختر اسم (مثلاً flight-erp) وكلمة سر لقاعدة البيانات → **Create**
3. استنى دقيقة لحد ما المشروع يجهز

## 2) إنشاء الجداول
1. من القائمة الجانبية افتح **SQL Editor**
2. انسخ محتوى ملف `database/schema.sql` بالكامل والصقه
3. اضغط **Run** — هتتعمل الجداول الأربعة بالفهارس

## 3) ربط التطبيق
1. من القائمة الجانبية: **Project Settings → API**
2. انسخ:
   - **Project URL** (شكله: https://xxxx.supabase.co)
   - **anon public key**
3. افتح ملف `lib/config/supabase_config.dart` والصق القيمتين مكان `YOUR_...`

## 4) تشغيل التطبيق
```bash
flutter pub get
flutter run
```

## ⚠️ ملاحظة أمنية
الجداول المنشأة من SQL Editor بتكون بدون Row Level Security.
ده مناسب للاستخدام الشخصي، لكن لو هتوزع التطبيق لاحقاً لازم نفعّل
RLS ونضيف نظام تسجيل دخول (هنعملها في مرحلة قادمة لو حبيت).
